Phase-1 Use case

  • User should be able to logIn/SignUp
  • Upon signup/login, the user will be directed to the home page, where he/she can browse writs that are already composed and published.
  • User can compose his/her own writ in the Compose page and publish it.
  • Upon publishing, the writ will get added to the home page carousel with the other writs.

DataBase Schema

  • Here, PenName, Password and EmailId will be provided by user at Signup event.
  • Unique UserId will be generated automatically upon successful signup.
  • Similarly, Content and Title will be provided by the user and Writ Id will be automatically generated
